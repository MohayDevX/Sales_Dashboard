s Performance Analysis Dashboard

هذا المشروع عبارة عن لوحة تحكم تفاعلية (Dashboard) لتحليل بيانات المبيعات، تم بناؤها باستخدام Python.

## محتويات المشروع:
- *تحليل الأرباح:* مقارنة أداء الولايات المختلفة.
- *تتبع الاتجاهات:* دراسة تطور المبيعات عبر الزمن.
- *تحليل الفئات:* توزيع المبيعات بناءً على فئات المنتجات.
- *تأثير الخصومات:* فهم العلاقة بين نسبة الخصم وصافي الربح.

## الأدوات المستخدمة:
- Python
- Pandas (لمعالجة البيانات)
- Matplotlib (للرسومات البيانية)

##الاستنتاجات:
يساعد هذا الداشبورد في اتخاذ قرارات مبنية على البيانات لتعظيم الأرباح وتقليل الخسائر الناتجة عن الخصومات العالية.
